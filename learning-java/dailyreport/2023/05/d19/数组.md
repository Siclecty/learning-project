### 数组

* 数组就是用来存储一批同种类型数据的容器。

* 数组适合做一批同种类型数据的存储。

#### 静态初始化数组

+ 定义数组的时候直接给数组赋值。

+ 数据类型[] 数组名 = {};

  #### 数组访问

- 数组名称[索引]
- 数组的长度属性：length
- 数组的最大索引：数组名. length – 1

#### 数组的动态初始化
+ 定义数组的时候只确定元素的类型和数组的长度，之后再存入具体数据。

+ 数据类型[]  数组名 = new 数据类型[长度];

#### 数组的遍历

+ 一个一个的访问数组中的数据。
+ 快捷键：数组名.fori。
